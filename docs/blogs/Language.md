# 변수와 상수 선언

변수 선언할 때 : var + 변수명: 타입
상수 선언할 때 : let + 상수명: 타입

    ```swift
    var i: Int
    let j: Character
    ```

# 초기화

    ```swift
    i = 1999
    j = "Swift"
    ```
    * 스위프트는 컴파일시 타입을 추론하여 최적의 타입을 지정한다.
    * Type Annotation 도 가능하다.

# 선언과 초기화

    ```swift
    var i = 1999
    let j = "Swift"
    ```
