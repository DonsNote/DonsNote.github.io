# 제 10 강 - 데이터 기반 의사결정과 전략적 활용

## 사전 질문
1. 데이터 기반 의사결정에서의 리스크는 무엇일까요? 이를 효과적으로 극복하기 위한 방법에는 어떤 것들이 있을까요?
    - 데이터 편향, 인과관계 오해, 맥락 무시 등의 리스크는 정량-정성 데이터 결합, 실험 설계(A/B 테스트), 다각도 분석, 그리고 비즈니스 맥락에 대한 깊은 이해를 통해 효과적으로 극복할 수 있음.
2. 제품팀이 균형 잡힌 의사결정을 내리기 위해 필요한 핵심 데이터 분석 역량은 무엇이며, 어떻게 개발할 수 있을까요?
    - 올바른 지표 설정, 데이터 해석과 스토리텔링, 통계적 사고, 기본 데이터 추출 능력이 핵심이며, 실제 제품 데이터로 반복 실습하고 크로스펑셔널 협업과 A/B 테스트 경험을 쌓으며 개발할 수 있음.
