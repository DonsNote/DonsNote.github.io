# 제어 전달문 (Control Statement)
* 제어 전달문은 코드의 한 부분에서 다른 부분으로 제어 흐름을 전달하여<br>
코드가 실행되는 순서를 변경해주는 구문입니다.
* 주로 반복문이나 조건문, 함수 등에서 사용됩니다.
